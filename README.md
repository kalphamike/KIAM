Summary: This is a chat-based portfolio website
It simulates a WhatsApp-style messaging interface to showcase projects and skills. Here's what it does:
Core Purpose:
A creative portfolio presentation that feels like chatting with the owner (Mike). Instead of a traditional static portfolio, visitors "chat" with the site to learn about projects.
Key Features:
1. Welcome Screen - Visitor enters their name to start the "conversation"
2. Project List (Left Panel) - Shows portfolio projects as chat threads:
   - KORA coaching (personal coaching platform)
   - UMWAShop (ecommerce demo)
   - Portfolio Overview
   - About Me
   - Inbox (for direct messages)
3. Chat Interface (Right Panel) - Each project shows:
   - Pre-populated messages explaining the project
   - Link to view the actual project
   - Phone/WhatsApp contact button
4. Status Feed - Instagram-like status stories showing project updates
5. Profile Tab - Shows owner's skills and info
6. Inbox - Allows visitors to send messages (gets auto-reply)
Design:
- WhatsApp green theme (primary color)
- Mobile-first responsive layout
- Desktop: two-column chat layout
- Mobile: bottom tab navigation
Use Case:
Creative way to present a portfolio - visitors engage in a "conversation" rather than passively reading. The inbox feature allows collecting visitor messages directly on the site.
